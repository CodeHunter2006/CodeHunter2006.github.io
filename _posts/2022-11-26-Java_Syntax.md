## import

- 关于**别名**
  Java 不支持在`import`时定义包或类的别名，如果有同名类需要同时使用，可以直接用包含全包名的类名。由于 Java 的包名就是域名的反向文本，所以不会出现全包名的冲突。

## 移位

- `>>>`
  无符号右移。由于 Java 中没有`unsigned`类型，所以在做移位操作时始终需要考虑符号位的处理。
  当最左边是`1`(负数)时，进行`>>>`操作时，最左边的`1`不会变化(不影响符号位)，符号位右边会自动补`0`。
